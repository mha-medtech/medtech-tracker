# MedTech Tracker 🏥

A web-based medical equipment tracking system designed for clinics and hospitals.
Built by a Biomedical Engineer with real-world experience in medical device regulation and management.

## 🌐 Live Demo
👉 [mha-medtech.github.io/medtech-tracker](https://mha-medtech.github.io/medtech-tracker)

---

## ✨ Features
- 📊 Real-time dashboard with equipment statistics
- 📈 Interactive doughnut chart (Chart.js)
- ⚠️ Calibration alert system — overdue and upcoming warnings
- ➕ Add, edit, and delete equipment records
- 🔍 Search and filter by status
- 🌙 Dark / Light mode toggle
- 🌍 Multilingual support (English / Deutsch / فارسی)
- 📱 Fully responsive — mobile & tablet friendly
- 💾 Data persistence via localStorage

---

## 🛠️ Tech Stack
- Frontend: HTML5, CSS3, JavaScript (Vanilla)
- Charts: Chart.js
- Backend *(coming soon)*: PHP
- Database *(coming soon)*: MySQL
- Deployment: GitHub Pages → Railway (planned)
